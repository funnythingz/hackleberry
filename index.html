hackleberry

HTML5/CSS3/JavaScript Framework

  • GitHub project
  • Version Dev.

What's this?

hackleberryはHTML5/CSS3/JavaScript によるスマートフォンに最適化されたUIを提供するためのフレームワークです。
CSSはSass/Compassを導入してハイパフォーマンスな開発を援助します。

Get Started

hackleberryの概要と、基本テンプレートを使ったチュートリアルを説明します。

1. Download

ソースファイルはダウンロードしてご参照ください。
このページでは、コンパイル版hackleberryファイルを使った作業の始め方のみご説明します。

作業を開始するのに最も簡単な方法です。 CSS、JS、画像が含まれたコンパイル縮小版で、余計なドキュメントやオリジナルのソースファイルは含まれていません。

全てのCSS、JavaScriptを含んだオリジナルのファイルです。 GitHubから直接、最新版をダウンロードすることが出来ます。

3. What's included

hackleberryはHTML、CSS、JSのあらゆるパーツを用意しています。 詳しくは各カテゴリページで説明しています。

Scaffolding

基本レイアウトやスタイルのリセット、余白、グリッドシステムを含んでいます。hackleberryを使う上では読み込み必須になります。

  • Layout
    • Container
  • Spacing
  • Grid system
    • Basic grid HTML
    • Offsetting columns
    • Nesting columns

Base Elements

ベースのHTML要素にスタイルをつけたものです。typography, lists, tables, forms, images が含まれます。

  • Typography
    • Headings & Text
    • Font & Alignment
    • Blockquote & Address
  • Lists
    • Marked & Numbered
    • Unstyled & Horizontal
    • Descriptions
  • Tables
    • Striped
    • Vertical
    • Bordered
    • & Hovered
  • Forms
    • Standard
    • Inputs
    • VerticalForm
    • HorizontalForm
  • Images
    • ImgRound
    • ImgCircle
    • ImgPolaroid

Components

hackleberryが独自に定義しているUIコンポーネント集です。ナビゲーション、タブ、ボタンなどを含みます。

  • Buttons
  • list-item
  • list-cassette
  • Tabs
  • Sprites

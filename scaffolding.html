Scaffolding

基本レイアウトやスタイルのリセット、余白、グリッドシステムを含んでいます。Hackleberryを使う上では読み込み必須になります。

Layout

基本レイアウトを構成する要素は.containerを使います。

Container

基本的なコンテナーをつくります。横幅を最小値/最大値が指定されています。

Spacing

.spacingで下方向にマージンがあきます。.spacing-leftで左方向、.spacing-rightで右方向にマージンがあきます。デフォルトで5pxあきます。

spacing

下方向にマージンがあきます。

text text text text text text text text text text text text text text

spacing*

下方向にマージンが10pxあきます。

text text text text text text text text text text text text text text

Grid system

Hackleberryでは100%幅のカラムを12カラムで分割したグリッドでレイアウトしています。1~12のカラムを使ってさまざまなレイアウトを構築できます。
.rowで枠をつくりその中の.col*でレイアウトします。また、カラムの間にマージンをとりたい場合は.offset*を使います。
